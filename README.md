# 🌾 Crop AI - Smart Agriculture Platform

A comprehensive, multi-modal Artificial Intelligence platform designed to empower farmers with Next-Generation agricultural solutions. This platform integrates Machine Learning, Natural Language Processing, Voice Commands, and Live Analytics to provide a 360-degree farming advisory system.

## 🌟 Key Features

1. **📈 AI Predictor (Dual-Engine)**
   - **Yield Prediction:** Uses a Random Forest Regressor to predict crop yields based on climate, soil pH, moisture, and N-P-K nutrient levels.
   - **Crop Recommendation:** Suggests the most profitable crop to plant based on your specific farm conditions using a Random Forest Classifier.
2. **🎤 Multi-Lingual Voice Assistant**
   - Hands-free control over the entire platform. Navigate pages, trigger predictions, and ask for crop information using your voice.
   - Supports English, Hindi, Tamil, and other regional languages.
3. **🤖 Deep-Context Agricultural Chatbot**
   - An intelligent chat assistant that remembers conversation history.
   - Provides expert advice on crop cultivation, localized pest/disease remedies, and fertilizer application.
4. **📊 Real-Time Dashboard & Live Ticker**
   - Simulated IoT sensor dashboard (Soil Moisture, Temperature, Nitrogen, pH).
   - Smooth, animated live market price ticker for major crops.
5. **🗺️ Live Agricultural Maps**
   - Interactive geographical maps to monitor farmland conditions.

## 🚀 Getting Started

### Prerequisites

Make sure you have **Python 3.8+** installed on your system.

### 1. Installation

First, clone or download this repository to your local machine. Navigate to the project directory:

```bash
cd "CROP YIELD PREDICTION 2"
```

Install the required Python dependencies:

```bash
pip install -r requirements.txt
```

*(Note for Voice Assistant: `pyaudio` and `speechRecognition` might require additional system-level audio dependencies depending on your OS. On Windows, `pip install pyaudio` usually works directly).*

### 2. Model Training (Optional but Recommended)

If the `models/` directory is empty or you want to generate fresh simulated data and retrain the Machine Learning models, run the training script:

```bash
python train_model.py
```
This script will:
- Generate 50,000+ rows of synthetic Yield, Recommendation, and Soil data.
- Train the `RandomForestRegressor` and `RandomForestClassifier`.
- Save the `.pkl` files inside the `models/` folder.

### 3. Run the Application

Start the Streamlit application by running:

```bash
streamlit run app.py
```

This will open a new tab in your default web browser (usually at `http://localhost:8501`).

## 🔧 Troubleshooting

- **Microphone Not Working in Voice Assistant:** Ensure your browser has granted microphone permissions to the `localhost` site. Check your system's default recording device.
- **Missing Module Errors:** Double-check that all packages in `requirements.txt` were installed successfully. 
- **PyAudio Installation Issues:** If `pip install pyaudio` fails on Windows, you may need to install the appropriate `.whl` file for your Python version from the Unofficial Windows Binaries for Python Extension Packages.
